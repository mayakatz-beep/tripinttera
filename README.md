# Trip in Tatra — מדריך טיול משפחתי

אתר סטטי בעברית למסלול המשפחתי ביולי 2026: בודפשט, מישקולץ, קושיצה, הרי הטטרה הגבוהים וברטיסלבה.

## מבנה הפרויקט

- `index.html` — האפליקציה המלאה: עיצוב, תוכן ו-JavaScript.
- `docs/integration-plan.md` — תוכנית העבודה לענף השדרוג.

## יכולות קיימות

- ניווט בין ימי הטיול והערים.
- רשימת הזמנות ורשימת אריזה עם שמירה מקומית בדפדפן.
- תחזית מזג אוויר, מפות, עלויות וקישורים.
- מצב כהה, שיתוף והדפסה ל-PDF.
- התאמה למובייל ותצוגת RTL.

## סביבת עבודה

האתר אינו דורש build או התקנת חבילות. לפתיחה מקומית אפשר לפתוח את `index.html` ישירות בדפדפן, או להריץ שרת מקומי:

```bash
python3 -m http.server 8000
```

ואז לפתוח `http://localhost:8000`.

## ענף העבודה

השינויים החדשים מתבצעים בענף:

```text
feature/slovakia-guide-integration
```

לאחר בדיקה ייפתח Pull Request מול `main`.
